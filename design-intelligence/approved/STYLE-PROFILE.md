## Como Atualizar este Perfil

Este arquivo e atualizado apos cada review (approve ou reject):

1. Agente loga a decisao em `APPROVED-DESIGNS.md` ou `REJECTED-DESIGNS.md`
2. Agente atualiza as tabelas de estatisticas acima
3. Se uma rejection tag atinge >= 2 ocorrencias, move para "Anti-Padroes Ativos"
4. Se uma approval tag atinge >= 2 ocorrencias, move para "Padroes de Sucesso Ativos"
5. Preferencias Aprendidas sao inferidas de padroes consistentes (3+ reviews concordantes)
